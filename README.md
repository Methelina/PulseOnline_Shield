## Реализация

### 3D-сетка

#### Полигональная сфера

- Создайте **икосферу** в **Blender** как минимум с 3 уровнями подразделения.
- Примените **модификатор Subdivision**.
- Выделите похожие вершины и удалите их (dissolve).
- Добейтесь, чтобы поверхность состояла только из шестиугольников и пятиугольников, наподобие футбольного мяча.

#### Развёртка UV

- Удалите стандартную развёртку UV и создайте новую с нуля.
- Это позволит сгруппировать грани с минимальными искажениями и максимальным перекрытием.
- Сгенерируйте UV-карту, содержащую как шестиугольные, так и пятиугольные формы.

#### Импорт в Unity

- Импортируйте готовую модель в Unity и настройте её для использования в проекте.

---

### Текстура

- Экспортируйте UV-развёртку из Blender.
- Используйте её как основу для создания текстуры контуров щита.
- Импортируйте текстуру в Unity и создайте на её основе материал без освещения (Unlit) для щита.

### Шейдер

#### Чистый HLSL

##### Shader Lab

- Установите **RenderType** и **Queue** в значение **Transparent**.
- Укажите, что шейдер предназначен для **UniversalRenderPipeline**.

```c
Tags {
    "RenderType" = "Transparent"
    "Queue" = "Transparent"
    "RenderPipeline" = "UniversalRenderPipeline"
}
```

- Включите **Cull Off** для отрисовки обеих сторон.
- Отключите **ZWrite**.
- Реализуйте **альфа-смешение**.

```c
Cull Off
ZWrite Off
Blend SrcAlpha OneMinusSrcAlpha
```

- Параметризуйте **основную текстуру**, **цвет** для внутренних граней и параметры **эффекта Френеля** на передней стороне.
- Задайте параметры для величины и скорости **анимации смещения**.
- Используйте **HDR** для цветов.

```c
_MainTex ("Texture", 2D) = "white" {}
[HDR] _ColorBack ("Color Back", Color) = (1,1,1,1)

_FresnelPower ("Fresnel Power", Float) = 1
[HDR] _FresnelColor ("Fresnel Color", Color) = (1,1,1,1)

_DisplacementAmount ("Displacement Amount", Float) = 1.0
_AnimationSpeed ("Animation Speed", Float) = 1.0
```

##### Вершинный шейдер

- Выполните смещение вершин в **этапе вершинного шейдера**.

```c
// смещаем грани вдоль нормалей
float displacementAmount =  ((sin(_Time.y * _AnimationSpeed) + 1) / 2) * _DisplacementAmount;
float3 displacedPostitionOS =  IN.positionOS.xyz + (IN.normal.xyz * displacementAmount);
OUT.positionHCS = TransformObjectToHClip(displacedPostitionOS);
```

- Вычислите **мировую нормаль** и **направление на камеру** для последующего расчёта **Френеля** во **фрагментном шейдере**.

```c
float3 positionW = TransformObjectToWorld(IN.positionOS.xyz);
OUT.viewDir = normalize(_WorldSpaceCameraPos.xyz - positionW.xyz);

OUT.worldNormal = TransformObjectToWorldNormal(IN.normal.xyz, true);
```

##### Фрагментный шейдер

- Рассчитайте эффект **Френеля**.

```c
// fresnelDot равен нулю, когда нормаль перпендикулярна направлению взгляда
float fresnelDot = dot(IN.worldNormal, IN.viewDir);

fresnelDot = saturate(fresnelDot);
float fresnelPow = pow(1.0f - fresnelDot, _FresnelPower);
```

- Верните соответствующий цвет, используя семантику **VFACE**, чтобы определить, обращена ли грань к камере.

```c
// VFACE положителен для передних граней,
// отрицателен для задних. Выводим один
// из двух цветов в зависимости от этого.
half4 frag(Varyings IN, half facing : VFACE) : SV_Target
{
    ...
    return facing > 0 ? color * fresnelPow * _FresnelColor : color * _ColorBack;
}
```

##### Шум Перлина

- Вычислите значение **шума Перлина**, используя координаты **normal.xy**.
- Благодаря тому, что все вершины одной грани имеют одинаковую нормаль (грань плоская), значение шума будет одинаковым для всех вершин грани.
- Умножив этот шум на величину смещения, мы заставим отдельные грани двигаться независимо друг от друга, создавая эффект хаотичного движения.

```c
// генерируем шум Перлина по UV второй развёртки
float noise;
PerlinNoise_half(
    IN.normal.xy,
    5,
    5,
    noise,
    _Time.y * _AnimationSpeed
);

// смещаем грани вдоль нормалей
float displacementAmount =  noise * _DisplacementAmount;
displacementAmount = clamp(displacementAmount, -_DisplacementAmount, _DisplacementAmount);

float3 displacedPostitionOS =  IN.positionOS.xyz + (IN.normal.xyz * displacementAmount);
```

#### Shader Graph

##### Цвет передней грани (Френель) и цвет задней грани

- Реализуйте **цвет задней грани** и **цвет передней грани с эффектом Френеля**.
- Используйте узел **Branch** и узел **Is Front Face**, чтобы выбрать нужный цвет.

##### Пользовательский узел функции для шума Перлина

- Примените узел **Custom Function** для вызова HLSL-функции шума Перлина.
- Используйте **координаты X и Y нормали в объектном пространстве** для вычисления шума.
- **Умножьте** значение шума на **DisplacementAmount**, затем **ограничьте** результат.

##### Смещение вершин

- **Сместите** **вершины** вдоль **нормалей** на вычисленную величину смещения.

### VFX Graph

- Установите пакет **Visual Effect Graph**.
- В **Shader Graph** включите опцию **"Support for VFX Graph"**.
- Настройте VFX Graph так, чтобы он порождал только одну частицу.
- Используйте узел **Output Particle Unlit Mesh**.
- Задайте **Size over Lifetime** для анимации появления/исчезновения.
- Настройте **Color over Lifetime** с помощью **Age over Lifetime** и **Sample Gradient**, передавая полученные значения в цвет материала щита.
